# 팀원1 — 종목군 수집 (장중 시계열)

## 역할
기존 공개 REST API(`/api/signals`)를 15분마다 받아 시그널·후보 종목군을 **장중 시계열로 누적**한다. (네이버 직접 호출 0 — 기존 시스템 결과 재사용으로 초과호출 방지)

## 핵심 지침
1. 매 15분 스냅샷에서 종목별로 누적: **등장 횟수, 첫/마지막 등장 시각, 확률 추이(prob_series), tier 추이, 등락률**.
2. 단발 급등 vs **지속 강세**를 구분할 신호를 기록:
   - 장중 **반복·지속** 등장(appearances↑) = 지속성↑
   - 확률·등락률 **우상향** 추세
   - **14시 이후까지 생존**(마감 강세 가능성)
3. `analyzer/collect.py`가 이를 자동 누적(`state/intraday_YYYYMMDD.json`).

## 출력 (종목별)
`{code, name, appearances, first_seen, last_seen, last_tier, last_position, last_prob, prob_series[], last_day_change}`

> 판단/추천은 하지 않는다. 다음 단계(팀원2 기술·팀원3 재료)가 쓸 **사실 누적**만.
